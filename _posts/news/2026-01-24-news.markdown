## 🤖 알고리즘 문제

**문제:** 문자열 배열에서 애너그램 그룹을 모두 찾아라

**설명:** 문자열 배열이 주어질 때, 애너그램(문자 조합이 같은 단어쌍)의 모든 그룹을 찾아 반환하세요.

**난이도:** Lv2

**입력 예시:** ["listen", "silent", "enlist", "google", "glegoo"]

**출력 예시:** [ ["listen","silent","enlist"], ["google","glegoo"] ]

**정답 (JAVA 8):**

```java
import java.util.*;
class Solution {
    public static void main(String[] args) {
        String[] words = {"listen","silent","enlist","google","glegoo"};
        Map<String, List<String>> map = new HashMap<>();
        for (String word : words) {
            char[] c = word.toCharArray();
            Arrays.sort(c);
            String key = new String(c);
            map.computeIfAbsent(key, k -> new ArrayList<>()).add(word);
        }
        for (List<String> group : map.values()) {
            if (group.size() > 1) System.out.println(group);
        }
    }
}
```
